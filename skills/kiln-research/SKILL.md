# Kiln Research

Research should narrow uncertainty, not inflate it.

## Deliverables

- concise findings tied to current project needs
- links or file references where appropriate
- updates to run docs only when the finding changes decisions

## Rules

- gather only what the next stage needs
- prefer primary sources when researching technical questions
- do not replace planning with endless reading
