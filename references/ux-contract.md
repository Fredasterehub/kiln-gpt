# UX Contract

Kiln should feel restrained, precise, and calm.

## Stage Transitions

Stage transitions should use a restrained visual rhythm:

- short stage label
- one-line narrative state
- no noisy banners or repeated ornamental framing

The effect should be intentional, not theatrical.

## Ambient Status

Default status output stays minimal:

- current stage
- current blockage, if any
- next safe point

Keep deeper detail in durable artifacts, not in constant chatter.

## Narrative Diagnostics

Diagnostic output should be:

- narrative-first
- path-backed
- technically precise

Always pair a concise explanation with the artifact or file path that supports it.

## Interruption Handling

The operator can interrupt naturally.

Rules:

- acknowledge the request briefly
- apply changes from the next safe point onward
- say what the safe point is when adaptation matters

## `tmux`

Recommend `tmux` for multi-agent visibility when useful.
Never make it a hard dependency.
